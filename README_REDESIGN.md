## 🔧 Customization

### Change Colors
```css
/* In style.css, modify :root variables */
:root {
  --primary: #0066FF;      /* Change primary blue */
  --accent: #00D9FF;       /* Change accent cyan */
  --text-dark: #2C3E50;    /* Change text color */
  /* ... etc */
}
```

### Add Sections
```html
<!-- Copy this template -->
<section id="new-section" class="section section--new">
  <div class="section-container">
    <div class="section-header">
      <h2>New Section</h2>
      <div class="section-divider"></div>
    </div>
    <!-- Your content here -->
  </div>
</section>
```

### Modify Spacing
```css
/* In :root, adjust spacing scale */
--spacing-lg: 3rem;  /* Larger sections */
--spacing-xl: 4rem;  /* More padding */
```
